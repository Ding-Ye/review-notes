# 工具效能

  • 开发环境（Mac）

    • VPN - 推荐配合 V2rayU 客户端使用
    • Mac Magnet -「💥」窗口管理工具，可在 AppStore 下载
    • Jetbrains IDE 产品族 ，「💥」推荐使用 toolbox-app 进行应用及项目管理。

  • 文档写作

    • PicGo -「👍」图床工具，低价开通 OSS 进行存储
    • Typora - Markdown 文档写作助手
    • Emoji 在线搜索
    • 中英文排版工具 - 让文档博客排版更清爽，Apple 官网采用的排版方式
    • PDF 在线转换

  • 个人工具

    • Xmind - 脑图工具
    • ProcessOn - 在线作图
    • Draw.io - 在线作图，各平台桌面也支持
    • Carbon - 代码生成图片
    • VuePress - 静态站点发布工具-相关资源

  • 开发工具

    • Java-Decompiler - Java 反编译工具，可查看 jar 、class 文件
    • JSON 在线编辑 -「👍」
    • Postman - API 开发
    • Arthas - Java 诊断工具
    • 正则表达式-可视化
    • 编程-在线为变量函数起名

  • Chrome 扩展插件

    • AdGuard 广告拦截器
    • Octotree - Github 侧导航插件，快速访问项目结构
    • Vimium vimium-快捷键 - vim 式的浏览器操作，释放鼠标
    • FeHelper -「👍」Web 前端助手，小工具集锦

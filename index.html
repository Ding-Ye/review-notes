# 快速开始

  • 为什么有这个项目？
    用于团队分享学习、复盘笔记资料共享

  • 在线阅读地址内容地址是？
    https://review-notes.top/

  • 网上学习资料那么多，为什么还写这些？
    因为网上是别人写的，我们阅读后知识吸收率太低，这个项目用于自身学习团队分享，主要使用费曼学习法进行，学习方法 。

  • 在线阅读内容怎么发布的？
    使用 vuepress 发布，使用 Travis CI 持续集成 ，交叉审核地址 http://dev.review-notes.top
